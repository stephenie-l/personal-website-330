Animation Project

The gifs for this project are created using different techniques. The Loop Dinosaur uses the for loop technique and has two rows of little monsters with blinking colours. The Jumping Egg uses the polar coordinates with noise and the outline remaining. The robot apocalypse reuses patterns and duplicates them. But instead of putting the drawing function in the setup like the example did, I put in in draw so the canvas doesn't refresh and it's actually like an apocalypse. The last three animations are the same series, with the weather changing from stormy to sunny rain. They use the same rain pattern but with different amounts of rain drops and different backgrounds.
